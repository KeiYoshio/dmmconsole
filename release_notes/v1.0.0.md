# Release Notes – v1.0.0

**Release date:** 2026-02-23

First stable release of **DMM Console**.

---

## Overview

DMM Console is a web-based control panel for digital multimeters.
Connect a DMM to your PC via GPIB, LAN, or USB and measure from any browser on the same network.

---

## Supported instruments

| Instrument | Interface | Note |
|------------|-----------|------|
| HP / Agilent 34401A | GPIB | Tested |
| Keithley 2000 | GPIB | Implemented, untested |
| Keithley 2010 | GPIB | Implemented, untested |
| ADCMT 7451A | GPIB | Tested |

## Features

- **Real-time measurement display** with automatic SI prefix scaling
- **Live waveform graph** – oscilloscope-like scrolling chart, selectable time window
- **Full instrument control** – function, range, integration time (NPLC)
- **CSV export** of the measurement buffer
- Runs in any modern browser (Chrome, Firefox, Brave) — no installation required on the client side

## Requirements

- Python 3.10+
- [agilent82357b](https://github.com/KeiYoshio/agilent82357b) v1.0.0 (for GPIB)
- Node.js (for frontend rebuild only)

## How to run

```bash
sudo uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

Then open `http://<server-ip>:8000` in a browser.

## Known issues

- **ADCMT 7451A USB direct mode** – write commands are silently ignored by the instrument.
  Root cause not yet identified. GPIB connection works correctly.
- Tested on macOS only. Linux expected to work. Windows untested.
