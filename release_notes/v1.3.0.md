# v1.3.0 – FY6800 DDS Signal Generator Support

## Summary

dmmconsole now supports the **FeelElec FY6800 DDS Signal Generator / Counter** via USB serial connection, adding signal generator control alongside the existing DMM functionality.

## New Features

### FY6800 Signal Generator Driver
- Two-channel output control (CH1 / CH2): 34 preset waveforms, frequency (up to 60 MHz), amplitude (0–20 Vpp), offset, duty cycle, phase, and output ON/OFF.
- Frequency counter with configurable gate time (1s / 10s / 100s).
- Auto-detection of FY6800 on serial ports.
- USB serial communication at 115200 bps via pyserial.

### Signal Generator Web UI (`/siggen`)
- Channel tabs (CH1 / CH2 / Counter) with channel-specific controls.
- Waveform selector with 34 preset waveforms.
- Numeric parameter inputs with Enter/blur-to-apply.
- Output ON/OFF toggle button.
- Gate time selector for counter mode.
- Real-time measurement display and waveform graph (counter mode).

### Serial Interface Support
- ConnectionBar now offers Serial as an interface option.
- Serial port auto-discovery via `GET /api/serial_ports`.
- Channel state sync via `GET /api/channel_state/{channel}`.

## Setup

1. Ensure `pyserial` is installed: `pip install pyserial`
2. Connect FY6800 via USB (CH340 serial adapter).
3. Navigate to `/siggen`, select the FY6800 model, choose Serial interface, select the port, and connect.

## Known Limitations

- Arbitrary waveform upload is not yet supported (DDS_WAVE handshake works, data format under investigation).
- Offset encoding is approximate (device-specific mapping needs refinement).
- Counter gate time readback (`RCT`) may return unexpected values on first read after power-on.
